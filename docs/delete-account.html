✦ Chordi 계정 삭제 안내

Chordi(개발자: hyeonji JANG)는 이용자가 언제든지 계정과 데이터 삭제를 요청할 수 있도록 안내합니다.

계정 삭제 요청 방법

이메일로 요청 — 아래 주소로 가입하신 계정(구글/카카오)의 이메일을 보내주세요.

📧 hyeonjijang@weknew.com (제목: "Chordi 계정 삭제 요청")

요청 확인 후 7일 이내에 계정과 모든 관련 데이터를 삭제해 드립니다.

삭제되는 데이터

  • 계정 정보 (이름, 이메일, 프로필)
  • 업로드한 악보·주보 사진
  • 작성한 곡·콘티 데이터
  • 푸시 알림 토큰

모든 데이터는 삭제 즉시 파기되며, 별도로 보관되는 데이터는 없습니다. 법령상 보관 의무가 있는 경우에만 해당 법령이 정한 기간 동안 보관 후 파기합니다.

일부 데이터만 삭제하고 싶다면

계정을 유지하면서 특정 곡·콘티만 삭제하려면 앱 안에서 해당 항목을 길게 눌러 삭제할 수 있어요. 직접 삭제가 어려운 데이터는 위 이메일로 요청해 주세요.
