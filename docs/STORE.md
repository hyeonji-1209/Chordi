# 플레이 스토어 등록 정보

## 앱 이름 (30자)
Chordi — 찬양팀 콘티

## 짧은 설명 (80자)
악보 사진 몇 장이면 AI가 콘티를 완성해요. 키 변경·코드차트·팀 공유까지 한 번에.

## 전체 설명

✦ Chordi는 교회 찬양팀의 한 주 준비를 사진 몇 장으로 끝내주는 앱이에요.

【주보 한 장으로 교회 등록】
교회 주보를 찍어 올리면 AI가 예배 목록을 읽고
주일 1부·2부, 수요예배, 금요기도회… 예배별 찬양팀을 한 번에 만들어 드려요.

【AI로 콘티 만들기】
악보 사진을 넣고 평소 말투로 적어 주세요.
"부르신 곳에서는 G키로 올려주고, 마지막 곡은 간주 없이 이어서"
키 변경, 곡 순서, 이어부르기까지 알아서 정리한 콘티가 완성돼요.

【연주 모드】
악보에서 뽑아낸 코드차트를 원하는 키로 즉시 이조해서 보여드려요.
송폼(Intro–V1–C…) 편집, 유튜브 음원 확인, 원본 악보 보기까지.

【팀과 함께】
초대코드·QR로 팀원을 초대하면 새 콘티가 올라올 때마다 푸시 알림이 가요.
팀원은 콘티 확인, 악보 보기, 키 확인만 하면 끝 — 준비는 인도자가, 공유는 자동으로.

찬양팀의 주중 단톡방 혼란을 이제 Chordi 하나로 끝내세요. ✦
